## Installation

L'application sera bientôt disponible via un **Test fermé** Android.
L’accès au test se fera via un questionnaire qui permettra de rejoindre le test, puis d’installer l’application depuis le Play Store.
L'application sera par la suite disponnible sur le PlayStore.

## Description

Silvernote est une application de prise de notes moderne, rapide et sécurisée.
Organisez vos idées avec des dossiers et des tags, sur tous vos appareils.
